## dCalendario
Tabela dimensão Calendário para Power BI Service

Dataflow gen1 Import (PRO) - Método indicado para os usuários do licenciamento PRO.

Benefício:
- SSOT;
- Padronização;
- Performance;
- Reaproveitamento.

Instruções:
- Criar um novo dataflow gen1;
- Criar uma consulta em branco;
- Copiar o código e colar no editor avançado da consulta em branco;
- Renomear tabela para dCalendario;
- Clicar em Salvar e fechar;
- Salvar o dataflow e dar o nome de dfCalendario;
- Atualizar o dataflow;
- Configurar a atualização automática diariamente às 00:00 de acordo com o fuso horário;
- Abrir um arquivo em branco no Power BI Desktop;
- Na guia Página Inicial Clique em Obter dados e depois em Fluxos de dados;
- Caso solicitado faça o login com sua conta e senha;
- Encontrar o Workspace e o Dataflow dfCalendario e marque a tabela dCalendario e clique em Carregar.
